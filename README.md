## 📂 Project Structure

This project consists of multiple interconnected static HTML pages styled uniformly with CSS and given interactivity with vanilla JavaScript.

- `index.html` - The main landing page / Home.
- `about.html` - Insight into my background, tech stack skills, education, and latest blog snippets.
- `project.html` - A gallery showcasing my past and on-going projects (Portfolio, Figma Designs, etc.).
- `services.html` - Details about the services I offer (UI/UX Design, Frontend Development).
- `contact.html` - A contact form page allowing clients or recruiters to reach out.
- `style.css` - The primary stylesheet containing all styling across the site (Dark theme, layouts, gradients).
- `script.js` - Contains interactivity logic, custom cursor setup, mobile menu configurations, particle effects, etc.

## 🛠️ Built With

- **HTML5** - Semantic and structured markup for all pages.
- **CSS3** - Custom styling, Flexbox, CSS Grid, custom properties (variables) for theme management.
- **Vanilla JavaScript (ES6+)** - Used for interactions such as the scroll animations, active link updates, custom cursor handling and mobile nav toggle.

## ✨ Features

- **Responsive Design:** Completely responsive layout adjusting smoothly from desktop to mobile screens.
- **Custom Theming:** Consistent dark-themed design with a vibrant primary accent color (`#b14eff`).
- **Interactive UI Elements:**
  - Dynamic Custom Cursor pointing over clickable elements (`script.js`).
  - Active navigation state changes on scroll.
  - "On Going" badges for in-progress project features.
  - Interactive "Bento Box" UI style on the About page.
